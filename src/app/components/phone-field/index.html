The phone field module contains a directive which validates phone numbers against the standards for different countries. The value that is validated against is driven through the ngModel attribute specified on the input element or a FormControl within a reactive form.

Specifies the default country for the phone field input and dropdown. This property accepts string International Organization for Standardization Alpha 2 values. Creates the phone field input and dropdown. Place this attribute on an input element, and wrap the input in a sky-phone-field component. Indicates whether to disable the phone field. This property accepts boolean values. Indicates whether to prevent validation on the phone field input. When set to true, date validation does not run on the input. This property accepts boolean values. This event emits a SkyPhoneFieldCountry object when the selected country is changed.
